This example takes information from the Input Data fields and sends it via an async fetch call to a /rest.api URI. The api handler simply takes the input data, some other request data, and adds a {status = 1} element then bundles it into a JSON response which is returned and displayed in the Response Body section below.

The URI can be the default "/rest.api" or optionally "/rest.api/item/NNN" where NNN is a positive integer value. In either case an optional query string can be added.
